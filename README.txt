To run this program you must compile the main file PixelInvaders.java
- Navigate the command prompt/terminal to the directory this repository 
is contained in
- type and run 'javac PixelInvaders.java' into the terminal, 
  this will compile all the necessary files 
- to run the program type 'java PixelInvaders' and hit Enter.
- The game window will open and you are ready to play
